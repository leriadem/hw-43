# HW - 43 My React Router App

Практична робота з налаштування маршрутизації у React за допомогою react-router-dom.
У застосунку реалізовано базову навігацію між сторінками: Головна, Про нас, Контакти.

## Запуск локально

1. Клонувати репозиторій:
   git clone https://github.com/leriadem/hw-43

2. Перейти в папку проєкту:
   cd hw-43

3. становити залежності:
   npm install

4. Запустити проєкт:
   npm start
